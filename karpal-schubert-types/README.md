# karpal-schubert-types

Schubert intersection type system for the Industrial Algebra ecosystem.

`karpal-schubert-types` begins Phase 14 of the Karpal roadmap:

- Schubert classes as type-level markers
- Intersection computation via Littlewood-Richardson coefficients
- Proof-carrying type assertions (`SchubertProven<λ, T>`)
- External verification of Schubert calculus
